## 如何使用

### 环境要求

- MaaFramework 运行环境
- mumu模拟器 设置平板 1280 720 可以尝试其他16:9分辨率 游戏内竖屏

### 安装使用

1. 从 [Releases](https://github.com/chenxing-ye/MAAFZHJ/releases) 下载最新版本
2. 解压到任意目录
3. 先尝试依赖安装
4. 运行主程序
5. 按照界面提示配置设备和任务
6. 开始执行任务
